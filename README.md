# Car-Price-Prediction-Model
A Regression based ML model for prediction of car prices in market based on `features of cars` and `dataset`

### Dataset Features : 

* Car Brand 
* Year 
* Sold Price
* Present Price
* Fuel Type
* Seller Type
* Transmission Type
* Owners
